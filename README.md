For jquery-practice
1. Pull the branch
2. Open in VS Code
3. download Live Server Extension for VS Code.
4. Run the questions-page.html

To add questions
1 add question in data/question.json with proper format (use the format use for first question)
2 add answers in data/answer.json  with proper format (use the format use for first  questions)
3 If question contains coding snippet use question key from question.json as file name with .txt extension
    and all characters should be in lowercase for file name. 
4 Save snippets over data/snippets/ here
